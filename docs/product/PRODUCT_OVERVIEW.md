# Product Overview

*   **One-Liner:** NomadWorks is an OpenCode plugin that installs the NomadWorks Collective, an AI-native software development team with specialized delivery roles.
*   **Target Audience:** Teams and solo builders who want AI agents to collaborate through explicit roles, task files, SCRs, and verification gates.
*   **Problem Statement:** Default agent workflows are fast but often inconsistent because one assistant is expected to behave like an entire engineering organization. NomadWorks solves that by introducing a structured AI software team with product, architecture, development, QA, review, and design roles that work through explicit artifacts and gated handoffs.

## Design Principles
- **Spec Before Delivery:** Significant changes should be grounded in approved SCRs and task artifacts.
- **Right-Sized Process:** Tiny work should stay lightweight, while complex work gets full orchestration.
- **Explicit Ownership:** Every task has a clear orchestrator, specialist handoff path, and verification gate.
- **Navigable Repos:** CodeMaps and documentation should make the repository understandable to future agents.
- **Team, Not Toolchain:** NomadWorks should feel like an AI software company embedded in the repository, with clear roles and disciplined collaboration.
